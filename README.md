## Tech

Tecnologias usada para este proyecto:

- [React.JS] - HTML enhanced for web apps!
- [Visual Studio Code] - Editor de texto

- [node.js] - para usar el manejador de paquetes de npm
- [Prettier] - para tener un buen formato de codigo
